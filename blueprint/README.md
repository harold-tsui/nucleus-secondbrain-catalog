# SecondBrain Public Blueprint

This directory is the concrete, git-trackable empty-shelf deliverable for
`nucleus-secondbrain-catalog`.

Copy this directory tree as the starting structure for a new SecondBrain vault,
then rename placeholder shelves such as `01-child`, `01-parent`,
`01-role-or-department`, and `01-method` to match your private instance.

Rules:
- Keep L1 domains stable.
- Create L2/L3 shelves only when they have a real purpose.
- Put private content in private repos or encrypted sync, especially family,
  health, finance, and personal-info shelves.
- Do not put real content in this public blueprint.

See `../BLUEPRINT.md` for the rationale and placement rules.
